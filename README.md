# 🎓 Pass or Fail Prediction Using Machine Learning and Python

![CI](https://github.com/mdshafi-786/PASS-OR-FAIL-PREDECTION-Using-MACHINE-LEARNING-AND-PYTHON-/actions/workflows/test.yml/badge.svg)

A complete, portfolio-ready ML project that predicts whether a student will **Pass or Fail** based on study habits and academic performance. Includes a Jupyter Notebook pipeline, Streamlit web app, CLI tool, and automated tests.

## Dataset Description

The dataset (`student_data.csv`) contains **600 synthetic student records** with realistic feature correlations:

| Column           | Range     | Description                                    |
|------------------|-----------|------------------------------------------------|
| `StudyHours`     | 0 – 12    | Number of hours the student studied per day    |
| `PreviousResult` | 30 – 100  | Score (%) from the previous exam               |
| `Attendance`     | 40 – 100  | Attendance percentage                          |
| `FinalMarks`     | 0 – 100   | Final exam marks (target variable)             |

**Pass/Fail threshold:** A student is classified as **PASS** if `FinalMarks >= 60`, otherwise **FAIL**.

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mdshafi-786/PASS-OR-FAIL-PREDECTION-Using-MACHINE-LEARNING-AND-PYTHON-.git
   cd PASS-OR-FAIL-PREDECTION-Using-MACHINE-LEARNING-AND-PYTHON-
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the notebook** (generates model files):
   ```bash
   jupyter notebook "PASS OR FAIL  PREDECTION.ipynb"
   ```
   Run all cells (Kernel → Restart & Run All).

## What is Tableau?

**Tableau** is a leading data visualization and business intelligence (BI) platform that allows users to connect to data sources, create interactive dashboards, and share insights — all without writing code.

### Key Features

| Feature | Description |
|---------|-------------|
| **Drag-and-drop interface** | Build charts, graphs, and dashboards visually |
| **Wide data connectivity** | Connects to CSV, Excel, SQL databases, cloud services, and more |

### Tableau vs. Python Visualization Libraries

| Tool | Best For |
|------|----------|
| **Tableau** | Interactive dashboards, business users, non-coders, quick visual exploration |
| **matplotlib / seaborn** | Static plots in notebooks and scripts |
| **Plotly** | Interactive charts embedded in Python apps |
| **Streamlit** | Quick Python-powered web apps with charts |

### Relevance to This Project

While this project uses **matplotlib**, **seaborn**, and **Streamlit** for visualizations, Tableau could be used as an alternative or complement to:
